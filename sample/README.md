# sample/

A **small, committed slice** of the corpus (e.g. ~5 policies / ~5 claims, all formats represented) so the
repo is usable and inspectable without running a full generation. Regenerated via `make sample` (seeded, so
it's stable). The **full** corpus is gitignored and published to HuggingFace — see [BRIEF.md](../BRIEF.md).

Populated in M2. Empty until then.
